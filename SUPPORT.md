# Support

Use the [Lumina issue tracker](https://github.com/FelixHenrikChristian/Lumina/issues)
for bug reports, feature requests, and usage questions.

Before opening an issue:

1. Check existing issues for the same problem.
2. Confirm the problem occurs on the latest Lumina release.
3. Include your Lumina version, Windows version, expected behavior, and actual
   behavior.
4. Add concise reproduction steps and screenshots when useful.

Do not attach private files, sensitive filenames, personal folder paths, credentials,
or security exploit details. Follow [SECURITY.md](SECURITY.md) for vulnerabilities.
